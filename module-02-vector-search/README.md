# Module 02 - Vector Search Homework

This project implements the official LLM Zoomcamp 2026 Module 02 homework as a reproducible Python project.

## Structure

- Homework/ contains the executable notebook, Python entrypoint, and supporting modules.
- The workflow follows the official homework instructions: load lesson pages, embed text with an ONNX embedder, perform vector search, text search, and hybrid search, and compute the homework answers programmatically.

## Quick start

1. Install dependencies with `uv sync`.
2. Run `uv run python Homework/homework.py`.
3. Open `Homework/homework.ipynb` in Jupyter.
